# Heemod cost-effectiveness model example

An example three state cost-effectiveness model in oncology built using
[`heemod::heemod-package()`](https://aphp.github.io/heemod/reference/heemod-package.html)
according to the assumptions and specification in the accompanying
paper.

## Usage

``` r
oncpsm
```

## Format

### `oncpsm`

A heemod object
